Pseudo setup guide, learning tool, memory burner for:

  • HTML5
  • CSS
  • JS

A beginner's cheatsheet arranged and annotated for no-bs construction of basic site files. Feel free to suggest edits on GitHub.

HTML5

Tag Usage Location Notes
<html> Document wrapper Initial parent Preceded by declaration <!DOCTYPE html>
<head> Undisplayed (meta) page info Within <html>, above <body>
<meta> Page attributes Within <head> Include http-equiv="Content-Type" content="text/html; charset=UTF-8"
<title> Browser tab page title Within <head>
<link> Linking stylesheets, scripts Within <head> Include rel="stylesheet" type="text/css"
<script> Inline scripts Within <head>, or end of <body>
<body> Page content wrapper Below <head>
<header> Leading content Below <body>
<nav> Navigation Within <header> Links setup as <ul> + <li>
<main> Banner image/text Within <header>
<section> Blocks of content Between <header> and <footer> Set unique id="name" attribute for nav
<div> One-off blocks of content Within <body>
<footer> Helpful page info End of <body>
Last updated July 2021.
