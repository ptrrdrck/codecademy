HTML5

Tag Place... Details Purpose
<html> Below initial <!DOCTYPE html> declaration. Include lang="en" attribute. Document wrapper
<head> Within <html>, above <body>. Undisplayed (meta) page info
<meta> Within <head>. Include http-equiv="Content-Type" content="text/html; charset=UTF-8" attributes. Page attributes
<title> Within <head>. Browser tab page title
<link> Within <head>. Include rel="stylesheet" type="text/css" attributes. Linking stylesheets, scripts
<script> Within <head>, or end of <body>. Inline scripts
<body> Below <head>. Page content wrapper
<header> Below <body>. Leading content
<nav> Within <header>. Links setup as <ul> + <li>. Navigation
<main> Within <header>. Good place for the <h1> and/or logo. Banner image/text
<section> Between <header> and <footer>. Set a unique id="name" attribute for navigation. Blocks of content
<aside> Between <header> and <footer>. Non-primary blocks of content
<div> Within <body>. One-off blocks of structure-content
<footer> End of <body>. Helpful page info
